ANN from Scratch

Description

Status: Finished


In early 2024, when I was learning reinforcement learning, I started learning the in's and out's of neural networks and decided I would try make an ANN from scratch. I built it to learn how perform the XOR gate operation, a classic problem in machine learning. The XOR gate is known for requiring a non-linear decision boundary, which makes it an ideal candidate for demonstrating the capabilities of a neural network with a hidden layer. I began by defining the structure of the network, which includes an input layer, a hidden layer, and an output layer. The hidden layer uses the sigmoid activation function, a smooth, non-linear function that squashes inputs to values between 0 and 1, enabling the network to learn complex patterns that would be impossible with a linear model.


I initialized the weights and biases of the network randomly to give the network a starting point for learning. Forward propagation was implemented to calculate the network’s output by processing the inputs through the layers, applying the weights, biases, and activation functions. The real challenge came in the backpropagation process, where I used the error between the predicted output and the actual XOR output to update the weights. This update was done by calculating gradients using the chain rule and adjusting the weights with gradient descent. The network’s learning process was guided by a cost function, specifically the Mean Squared Error (MSE), which I minimized during training to improve the network’s accuracy.


The learning rate controlled the size of the updates to the weights, balancing the speed and stability of the training process. Over 5000 iterations, the network repeatedly adjusted its weights, gradually improving its ability to correctly perform the XOR operation. The project culminated in a fully trained neural network capable of correctly identifying the XOR output for any given pair of inputs, demonstrating the power and flexibility of neural networks in solving non-linear problems.
